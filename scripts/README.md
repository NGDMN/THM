# Organização dos arquivos CSV

Todos os arquivos CSV gerados ou utilizados pelos scripts agora ficam na pasta `data/`.

- `data/municipios_RJ_SP.csv`: lista de municípios do RJ e SP
- `data/municipios_RJ_SP_coords.csv`: municípios com coordenadas
- `data/previsoes.csv`: previsões meteorológicas
- ...

Certifique-se de sempre utilizar e salvar arquivos nesta pasta para manter o projeto organizado.
